Strukture grafovskih baz

Primerjava med Property Graph in RDF (Semantic) modelom.

Property Graph

Vozlišča in povezave imajo lastnosti (ključ-vrednost).
Janez
starost: 30
mesto: MB
Marija
vloga: Admin
POZNA {od: 2020}

RDF Graph (Triples)

Vse je trojček: (S) -> [P] -> (O). Brez notranjih lastnosti.
:Janez
:pozna
:Marija
:starost
"30"^^xsd:int
:mesto
"Maribor"
Ključne razlike:
  • Property Graph: Bolj intuitiven za razvijalce aplikacij, bogati podatki so "skriti" znotraj vozlišč.
  • RDF: Osnova za Semantic Web. Vsaka lastnost je svoja povezava, kar omogoča globalno povezovanje podatkov (interoperabilnost).
