VECTOR CALCULUS FOR GAMES

1. Gradient (∇f) - Terrain Following

Kaže v smer najstrmejšega vzpona. Uporablja se za AI, ki se želi povzpeti na hrib, ali za izračun normal terena.
∇f = (∂f/∂x, ∂f/∂y) | Agent sledi beli puščici "navzgor".

2. Curl (∇×F) - Rotational Flow

Meri lokalno rotacijo v vektorskem polju. Ključno za simulacijo vrtincev, dima in zračnih tokov.
∇×F = (∂Fy/∂x - ∂Fx/∂y) | Večji ko je rotor, hitreje se vrti objekt.

3. Divergence (∇·F) - Source/Sink

Meri, ali se polje v točki "širi" (izvir) ali "krči" (ponor). Osnova za nestisljive tekočine.
∇·F = ∂Fx/dx + ∂Fy/dy | Pozitivna = izvir, Negativna = ponor.

4. Laplacian (∇²f) - Field Smoothing

Meri razliko med točko in povprečjem okolice. Uporablja se za erozijo terena in simulacijo toplote.
∇²f = ∂²f/∂x² + ∂²f/∂y² | Uporabljeno za glajenje robov.
