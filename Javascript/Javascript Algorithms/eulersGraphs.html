Eulerjeve Poti in Cikli

Graf ima Eulerjev cikel, če so vsa vozlišča sode stopnje (parno število povezav).

1. Eulerjev Cikel (Vsa sode stopnje)

Vsako vozlišče ima stopnjo 2 ali 4. Lahko se vrnemo v izhodišče.

EULERJEV CIKEL

2. Eulerjeva Pot (2 lihi stopnji)

Dve vozlišči imata liho stopnjo. Pot se začne v enem in konča v drugem.

EULERJEVA POT

3. Ni Eulerjev (4 lihe stopnje)

K4 graf (popoln graf s 4 vozlišči). Vsa vozlišča imajo stopnjo 3.

NI EULERJEV

4. "Hišica" (Eulerjeva Pot)

Znan problem risanja brez dviga pisala. Spodnja vozlišča so liha.
