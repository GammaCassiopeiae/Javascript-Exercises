Description

Simple sorting algorithm that repeatedly steps through the list, compares adjacent elements and swaps them if they are in the wrong order.

Complexity

Best:Ω(n)
Average:Θ(n²)
Worst:O(n²)
Space:O(1)

Pseudocode

for i from 0 to n-1
  for j from 0 to n-i-1
    if arr[j] > arr[j+1]
      swap(arr[j], arr[j+1])
