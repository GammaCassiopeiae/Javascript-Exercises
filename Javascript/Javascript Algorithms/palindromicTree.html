Advanced Data Structures Lab

1. Palindromic Tree (Eertree)

Struktura za iskanje vseh unikatnih palindromov v nizu.

2. Ternary Search Tree

Kombinacija Trie in binarnega drevesa za shranjevanje nizo.

3. Interval Tree

Iskanje prekrivajočih se intervalov v času O(log n).

4. BK-Tree (Fuzzy Search)

Iskanje besed na podlagi Levenshteinove razdalje.

5. Cartesian Tree

Binarno drevo, zgrajeno iz zaporedja (Heap property).

6. Sparse Set

Konstantni čas O(1) za dodajanje, iskanje in brisanje.

7. Persistent Stack

Shranjevanje vseh prejšnjih različic strukture.

8. Gomory-Hu Tree

Reprezentacija vseh min-cutov v grafu.

Gomory-Hu Tree reducira n(n-1)/2 max-flow problemov na n-1.
