What is Python?

Python is a high-level, interpreted, general-purpose programming language created by Guido van Rossum and first released in 1991. Python emphasizes code readability with significant indentation and supports multiple programming paradigms including procedural, object-oriented, and functional programming.

[03] INPUT AND OUTPUT

Output with print()

The print() function outputs data to the console. 

# Basic print
print("Hello, Python!")

# Print multiple values
print("Name:", "Alice", "Age:", 25)

# Formatted output with f-strings
name = "Bob"
age = 30
print(f"{name} is {age} years old")

# Custom separator and end
print(1, 2, 3, sep="-", end="!\n")

Input with input()

The input() function reads user input as a string. 

# Basic input
name = input("Enter your name: ")
print(f"Hello, {name}!")

# Converting input to numbers
age = int(input("Enter your age: "))
height = float(input("Enter height (m): "))

# Input with validation
while True:
    user_input = input("Enter a number: ")
    if user_input.isdigit():
        break
    print("Invalid input!")

[04] VARIABLES

Variable Declaration

Python variables are created when you assign a value to them. No explicit declaration needed. 

# Variable assignment
x = 5
y = "Hello"
pi = 3.14159
is_active = True

# Multiple assignment
a, b, c = 1, 2, 3

# Same value to multiple variables
x = y = z = 0

# Swapping values
a, b = b, a

# Variable naming rules
user_name = "john"  # snake_case convention
MAX_VALUE = 100   # Constants in UPPERCASE
_private = 42      # Private by convention

Variable Types & Type Checking

# Dynamic typing
x = 5           # int
x = "hello"     # now str
x = 3.14        # now float

# Type checking
type(x)          # Returns: <class 'float'>
isinstance(x, float)  # Returns: True

# Type conversion
num_str = "123"
num_int = int(num_str)
num_float = float(num_str)
text = str(456)

[05] OPERATORS

Arithmetic Operators

a, b = 10, 3

print(a + b)  # Addition: 13
print(a - b)  # Subtraction: 7
print(a * b)  # Multiplication: 30
print(a / b)  # Division: 3.333...
print(a // b) # Floor Division: 3
print(a % b)  # Modulus: 1
print(a ** b) # Exponentiation: 1000

Comparison & Logical Operators

x, y = 5, 10

# Comparison
print(x == y)  # Equal: False
print(x != y)  # Not Equal: True
print(x > y)   # Greater: False
print(x < y)    # Less: True
print(x >= 5)  # Greater or Equal: True

# Logical
print(True and False)  # AND: False
print(True or False)   # OR: True
print(not True)         # NOT: False

Assignment & Identity Operators

x = 10
x += 5   # x = x + 5 → 15
x -= 3   # x = x - 3 → 12
x *= 2   # x = x * 2 → 24
x //= 4  # x = x // 4 → 6

# Identity operators
a = [1, 2]
b = [1, 2]
c = a

print(a is b)    # False (different objects)
print(a is c)    # True (same object)
print(a == b)    # True (same values)

[08] CONDITIONAL STATEMENTS

If-Elif-Else Statements

age = 18

if age < 13:
    print("Child")
elif age < 20:
    print("Teenager")
else:
    print("Adult")

# Ternary operator
status = "active" if age >= 18 else "inactive"

# Multiple conditions
score = 85
if score >= 90:
    grade = "A"
elif score >= 80 and score < 90:
    grade = "B"
else:
    grade = "C"

# Truthy/Falsy values
if []:        # False (empty list)
    print("Won't print")
if [1]:      # True (non-empty)
    print("Will print")

Match-Case (Python 3.10+)

day = "Monday"

match day:
    case "Monday":
        print("Start of week")
    case "Friday":
        print("TGIF!")
    case "Saturday" | "Sunday":
        print("Weekend!")
    case _:
        print("Weekday")

[09] LOOPS

For Loop

# Iterate over sequence
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

# With index using enumerate()
for i, fruit in enumerate(fruits):
    print(f"{i}: {fruit}")

# Using range()
for i in range(5):
    print(i)  # 0 to 4

# Nested loops
for i in range(3):
    for j in range(3):
        print(f"({i},{j})", end=" ")

While Loop

# Basic while loop
count = 0
while count < 5:
    print(count)
    count += 1

# With else clause
num = 0
while num < 3:
    print(num)
    num += 1
else:
    print("Loop completed!")

# Infinite loop with break
while True:
    user_input = input("Enter 'quit' to exit: ")
    if user_input == "quit":
        break

Loop Control: Break, Continue, Pass

# Break - exit loop
for i in range(10):
    if i == 5:
        break
    print(i)  # 0 to 4

# Continue - skip iteration
for i in range(5):
    if i == 2:
        continue
    print(i)  # 0, 1, 3, 4

for i in range(5):
    pass  # TODO: implement later
