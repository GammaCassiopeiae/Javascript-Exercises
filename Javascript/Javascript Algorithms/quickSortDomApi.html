How It Works

QuickSort selects a pivot element, then partitions the array so all elements smaller than the pivot are on its left, and all larger on its right. It then recursively sorts the two sub-arrays. This visualizer uses the last element as the pivot (Lomuto partition scheme).

Complexity

Best / Average: O(n log n)
Worst case: O(n²) — occurs when the pivot is always the smallest or largest element.
Space: O(log n) — recursion stack depth.
In-place: Yes  |  Stable: No
