Suffix Array & Suffix Tree

1. Suffix Array (Tabela pripon)

Leksikografsko urejen seznam vseh pripon niza. Uporabno za binarno iskanje podnizov.

Urejene pripone:
    Ukkonen’s Algorithm (Part 1):
    Ukkonenov algoritem gradi Suffix Tree v času O(n) z uporabo "online" pristopa (dodaja znak po znak). Ključni so "suffix links", ki omogočajo hiter preskok med vejami.

    2. Iskanje podniza (Tree Application)

    Generalized Suffix Tree:
    Če združimo več nizov (npr. S1# in S2$), dobimo posplošeno drevo, ki omogoča iskanje najdaljšega skupnega podniza med dvema dokumentoma.
